# SeqMaker
Video to JPEG Image Sequence converter, automatic, drag and drop. Converts videos to JPEG sequences at various frame rates (just add folder labeled with fps outputs desired, ie; 1fps, 2fps, 3fps, etc). Auto labels folder with the video file name + frame-count (video-file-name_#-of-frames). Embeds customizable copyright metadata, moves processed files to a done/ folder. Runs anywhere, Desktop or external drives. Double-click start/stop. Dowloadable script is in .zip form.
